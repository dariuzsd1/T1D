# T1D Supply Hub

[![CI](https://github.com/dariuzsd1/T1D/actions/workflows/ci.yml/badge.svg)](https://github.com/dariuzsd1/T1D/actions/workflows/ci.yml)

An agent and web app that takes the load off living with Type 1 Diabetes — track medical supplies, forecast refills, and stay ahead of shortages without the stress.

The guiding goal is reducing mental load: helping people remember supplies, prescriptions, expiration dates, insurance/refill timing, and device replacements so they spend less time thinking about diabetes.
